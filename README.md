# Построитель графиков функций на Android
Оно полностью написано на Python, поэтому у него такой относительно большой размер ;)
Приложение способно строить графики функций (например, sin(x)), для этого необходимо ввести функцию в поле для ввода и нажать "построить". 
Писать в начале "y=" или "f(x) =" не надо. 
Если вы хотите изменить видимую область построения, то измените соответствующие значения в предназначенных для этого полях и снова нажмите кнопку "построить". 
Вроде все написал.
