# Построитель графиков функций для Android

Приложение полностью написано на Python, поэтому имеет относительно большой размер ;)

Оно способно строить графики функций (например, `sin(x)`). Для этого необходимо ввести функцию в поле ввода и нажать **«График»**.  
Писать в начале `y=` или `f(x)=` **не нужно**.

В **параметрическом режиме** параметром является `t`. Если использовать вместо `t` любой другой символ, график **не будет отрисовываться**.  
В этом режиме **обязательно заполнить оба поля**, иначе график не появится!

В **обычном режиме**, помимо функций вида `x - 5`, `x^2 - 2x` и подобных, можно использовать параметры: например, `a*x + b`.  
После нажатия кнопки **«График»** появятся слайдеры для изменения значений введённых параметров.  
⚠️ **На параметр `t` это не распространяется!**

Если вы хотите изменить видимую область построения, измените соответствующие значения в полях диапазонов и снова нажмите **«График»**.
